Docker
=========
A generic role made for installing docker on multiple platforms.

Current platforms:
- Linux
  - CentOS 8
  - Fedora 32
  - Ubuntu 20.04
  - ArchLinux

Example Playbook
----------------

```yml
- hosts: all
  roles:
      - docker
```
