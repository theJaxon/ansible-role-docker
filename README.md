Docker
=========
Install docker on Multiple platforms.

Current platforms:

|     Name    |    Version    |
|:-----------:|:-------------:|
| RHEL/CentOS |       8       |
|    Fedora   |       32      |
|    Ubuntu   | Focal (20.04) |
|  Archlinux  |      all      |

Example Playbook
----------------

```yml
- hosts: all
  roles:
      - docker
```
